File Handles


Always use a handle
when using a file
National Safety Council of Australia

You should always use a file handle when using a file. Accidents happen quickly, and as noted in the picture to the right, a file handle could help prevent you getting significant damage to your hand, or at least minimize the injury.

File handles are not expensive; they are significantly less cost than a visit to the emergency room.

Wooden File Handle

Wooden File Handles

These can often be found at used tool sales: garage sales, flea markets, etc. Some online sellers and some hardware stores carry them.

If you are a wood turner, they are easy to make, and the ferrule can be made using a piece of copper tubing.

Many of these have a mechanism where the file's tang is "screwed" into the handle. Other types have the file's tang pressed into the handle and the file is held using friction.

I'm a fan of the screw-in approach, and the brand I use is Scroo-zon from JTS Lutz. These are easily found with a quick search on the Internet.

Plastic File Handle

Plastic File Handles

These have the file's tang pressed into the handle and the file is held using friction.

Hardware stores and big-box vendors carry them, as do many online vendors.

Husqvarna's Chainsaw File Handle

Chainsaw File Handles

A spring-loaded part in the center of the handle is pressed down, and then the file's tang is pressed into the handle.

Hardware stores and many big-box vendors carry them, as do many online vendors.
