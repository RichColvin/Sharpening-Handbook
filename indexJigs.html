Jigs, Fixtures, & Modifications for the Tormek

  

Some really great add-ons have emerged for the Tormek. This page contains a quick reference to those user-designed jigs, fixtures, and modifications.

Sharpening Drill Bits

  • DBS-22 Modifications - These modifications to the DBS-22 make it easier to use, and expand the capabilities of the DBS-22.
  • USB Spacer for the DBS-22 - for faster setup of the Tormek when sharpening twist drill bits.

Sharpening Knives

  • Front Vertical Base (FVB) - for faster and more consistent setup when sharpening knives
  • Jigs used for faster and more consistent setup when sharpening knives
    • HanJig
    • KenJig
    • Knife Angle Setter
  • Laser Goniometer - for identifying the angle that is on a knife, so the grind angle can be matched
  • Platform for Honing and Sharpening Knives - for easier work when sharpening multiple knives
  • Rotary Knife Jig - for sharpening the blade on the rotary knives which seamstresses use

Sharpening Multiple Types of Tools

  • Goniostat - for sharpening tools at specific angles
  • Pivot Jig for SVM-45 Knife Jig - for making it easier to keep the grind angle consistent along the end of the blade as it curves to a point. Very useful for sharpening pruning shears for gardening.

Sharpening Woodworking Bench Tools

  • Alignment Jig for the SE-77 - for setting the SE-77’s two adjustment screws for proper alignment.
  • Projection Jig for the SE-76 or SE-77 (Used with the TTS-100) - for setting the projection of the tool when using the SE-76, SE-77, or SVD-186.
  • Router Plane Blade Jig - for holding the traditional router plane blade when sharpening it, ensuring best practices for aligning it properly.
    • Woodpeckers Router Plane Blade Jig - for holding the removable blades used on Woodpeckers planes.

Other

  • AngleGauge - A simple gauge for identifying angles on tools
  • Break-in Jig - This is used for new diamond or CBN grinding wheels to “break” them in, reducing the aggressiveness seen on new grinding wheels.
  • Drip Tray - This is used to send water, dripping from tools, back into the water trough (instead of dropping onto the top of the machine).
  • Honing Support Bar - a secondary bar which allows for sharpening and honing without having to re-adjust the USB when switching between the two wheels
  • Stone Nut Spacer - for keeping the spindle aligned properly when the grindstone has been removed.
