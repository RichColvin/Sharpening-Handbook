Low Cost Shop Computer

The default system used on a Raspberry Pi is Raspbian, a computer operating system based on a variant of Linux called Debian.

Raspbian works well enough, but there is now a far better option : Ubuntu MATE (pronounced "ma" "tay"). Below are the instructions for getting and installing that on your Raspbery Pi.

Step 1 - Get Ubuntu MATE for the Raspberry Pi
  1. Download the Ubuntu MATE image.
    1. On the Ubuntu MATE web site, click on Download at the top of the screen, then select Raspberry Pi.
    2. You can try the experimental stuff if you like, but I opt for the stable version.

    Micro SD Cards

  2. Once the image file is downloaded, you will need to load it onto the Micro SD card. The Micro SD card is in the kit with the Raspberry PI, and is tiny.

    Plug the Micro SD card into the adapter (e.g., the IOGear SuperSpeed USB 3.0 SD / MicroSD Card Reader / Writer), and then connect the adapter to your computer.

    Note: Micro SD cards will only insert fully one way.

    • If it is inserted properly, it will push in quite easily. No force is necessary.
    • If it is inserted upside down, it will not seat properly.

  3. Extract the downloaded file onto the Micro SD card. Depending on what computer you are using :
    • Mac OSX
      1. Download and install Etcher.
      2. Select the image file you downloaded earlier and the location of the SD card.
      3. Click "Flash" and watch the pretty progress bar. This will take a 20 minutes or so. But be patient: it is worth the wait.

    • Windows
      1. Download and install Win32DiskImager.
      2. Select the image file you downloaded earlier and the drive letter of the SD card. Warning: There is a significant risk you could damage your file system if you select the wrong drive letter. Make sure you get it right!
      3. Click "Write" and watch the pretty progress bar. This will take a 20 minutes or so. But be patient: it is worth the wait.

Step 2 - Assemble the Raspberry Pi

Put all the physical stuff together. Don't plug in the power until last.

  1. Put the Raspberry Pi board into the case.
  2. Insert the Micro SD Card into the slot on the end of the Raspberry Pi board.
  3. Plug the monitor cable into the HDMI jack. Turn the monitor (or TV) on.
  4. Plug the keyboard cable into one of the USB jacks. It doesn't matter which one.
  5. Plug the mouse cable into one of the USB jacks. It doesn't matter which one.
  6. If you are using a wired connection to the netwirk, plug the ethernet cable into the ethernet jack.
  7. Plug the power into the power jack, and plug the other end into the electrical outlet.

If you want to see a video of the assembly, you can watch it on the YouTube video below. It starts about 2:00 into the video, and goes for about a minute. (There is more here about installing Raspbian, but you can skip that.)

Step 3 - Setup Ubuntu MATE - Part 1

Now let's power everything on, and get Ubuntu MATE up and running. Once the Raspberry Pi gets powered on, it will go thru some screens to ask you questions needed for setting it up for you.

  1. Welcome
    1. Select the language you want to use. For me, I am using English.
    2. Click Continue.

  2. Keyboard layout
    1. Select the keyboard you want to use. For me, I am using English (US).
    2. Click Continue.

  3. Where are you?
    1. Enter the city where you live, or the closest one it can find. This is to set your computer's timezone.
    2. Click Continue.

  4. Who are you?
    1. Enter your name. I use my first and last name. (You can change this later if you wish.)
    2. Give your computer a name. You can use the default it gives, or give it something unique. I try to make it something descriptive so that when I see the list of machines on my network, I know what it is: for example, "Rich-RaspberryPi-01".
    3. Enter what you want to use for a user name. I use my first name, and I use all lower case letters. This is pretty standard in the Linux community.
    4. Enter what you want to use for a password. Write it down so you don't forget it.
    5. Key Note: be sure to remember your password. There is no way to reset it, and you'll have to start over if you forget it.
    6. Click Continue.
It's going to run for a while now. Loads of stuff is happening. But suffice it to say, you certainly have time to go get a cup of coffee.

Step 4 - Setup Ubuntu MATE - Part 2

After Step 3 finishes, the system will reboot itself. Logon with the user name and password you gave above in Step 3.

  1. Reboot again.
    1. On the top right hand of the screen, there is a gear icon. Click on this.

    2. In the pull-down menu that appears, click Shutdown ..., and then click Restart

    Logon again to continue ...

  2. Check / setup your network connection
    1. If your network is wired, nothing need be done.
    2. If your network is Wi-Fi :
      1. On the upper right part of the screen is an icon that is an up/down arrow ↑↓. Click on this.

      2. Click on your network below WiFi Networks. You will then enter your network access code.
    3. Test to be sure it works by clicking on the Firefox browser icon which is on the screen in the upper left, and going to a web site not shown. I recommend something like www.Google.com.

  3. Install the latest set of patches
    1. A Welcome menu is in the middle of the screen. Click on the Software button. This is brings up the Software Boutique.
    2. Click on the check box near the top of the window. This is for Fixes
    3. Click on Update Sources List. Let that finish.
    4. Click on Upgrade Installed Packages. Let that finish (it will run 15 mins or more, and you may have to answer a question or two along the way).
    5. Scroll down, then click on Configure Interrupted Packages. Let that finish.
    6. Click on Resolve Broken Packages. Let that finish.

  4. Reboot again.
    1. The system may tell you it needs to restart. If it does, take that option.
    2. If it does not ... On the top right hand of the screen, there is a gear icon. Click on this.

      Then, in the pull-down menu that appears, click Shutdown ..., and then click Restart

      3. Logon again to continue ...

    3. Click on Menu at the top left of the screen

      then select Control Center in the pull-down menu that appears.

    4. Click on Software Updater.
    5. You will get a pop-up that there are updates. Update the system as recommended.
    6. Once that finishes, you will probably have to re-boot the system. If you are not requested to reboot, do so anyway.

  5. Setup for automatically patching in the future
    1. Click on Menu at the top left of the screen

      then select Control Center in the pull-down menu that appears.

    2. Click Software and Updates
    3. On the Ubuntu Software tab, enable :
      1. Canonical-supported free and open-source software (main)
      2. Community-maintained free and open-source software (universe)
      3. Software restricted by copyright or legal issues (multiverse)
    4. On the Updates tab, enable :
      1. Important security updates (bionic-security)
      2. Recommended updates (bionic-updates)
    5. Click Close, then answer any questions that may appear.
    6. A window will popup saying "The information about available software is out-of-date". Click Reload. Let this finish.

  6. Setup NTP to keep your time correct.
    1. In the Welcome menu that is in the middle of the screen, click on the Software button. This is brings up the Software Boutique.
    2. In the Software Boutique window, open the search by clicking on the magnifying glass in the upper right of the window. In the search field, enter "NTP", then depress the Enter key.
    3. When you see the message saying "No software matched your key words.", Click on Install a software center.
    4. Take the upper option for Software. Click the green button labeled Install. button, then the Apply Changes button. You will have to enter your password to confirm that you want to do this. Once it completes, click Close.

    5. Click on Menu at the top left of the screen

      then select System Tools in the pull-down menu that appears, then Software.

    6. Click the magnifying glass at the top of the window, then enter "NTP" and depress Enter.
    7. Click on ntpcontrol, then click on the Install buttion. You will have to enter your password to approve the installation of this software.
    8. Click Launch.
    9. Close the window.

    10. Click on Menu at the top left of the screen

      then select System Tools in the pull-down menu that appears, then MATE Terminal.

    11. Enter the command snap connect ntpcontrol:timeserver-control, and depress the Enter key. Enter your password.
    12. Enter the command sudo ntpcontrol set ntpserver tick.usno.navy.mil, and depress the Enter key.

      13. tick.usno.navy.mil is the NTP server at the U.S. Naval Observatory. It is my preferred go-to clock.

    13. Enter the command sudo reboot, and depress the Enter key.

  7. Secure the Sytem - Turn on the Firewall
    1. This is optional, but I recommend doing this.
    2. Click on Menu at the top left of the screen

      then select Control Center.

    3. Click on Firewall Configuration.
      1. Set the Status to On.
      2. Set the Incoming to Deny.
      3. Set the Outgoing to Allow.
    4. Close the window.

  8. Secure the Sytem - Disable Bluetooth
    1. This is optional, but if you don't need it I recommend doing this. It wastes computing power, and enables ports of entry for hackers that you don't need.
    2. To turn it off, click on the bluetooth icon on the upper right part of the screen. Then in the pull down window, click on Turn Bluetooth Off

  9. You're Done !
    1. Enjoy your low cost shop computer !
    2. Oh, and by the way, once or twice a year, you should reboot it just for good measure.
