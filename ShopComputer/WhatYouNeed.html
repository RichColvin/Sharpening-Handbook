What You will Need

There are quite a few models of the Raspberry Pi, but I use the Pi 3 Model B+. If you need help identifying the connectors mentioned below, here is a quick reference.

  1. Raspberry Pi Kit, like this one from MicroCenter : CanaKit Raspberry Pi 3 Model B+ Complete Starter Kit. This is around $75.

  2. Adapter to connect a Micro SD card to your computer, like this one from MicroCenter : IOGear SuperSpeed USB 3.0 SD / MicroSD Card Reader / Writer. This is around $15-20.

  3. Computer keyboard with a USB connector.

  4. Computer mouse with USB connector.

  5. Ethernet cable (if you are hooking the machine up to a jack -- the Raspberry Pi has WiFi capabilities that you could use instead)

  6. Computer Monitor, or a TV with HDMI input(s). The Raspberry Pi uses HDMI for video output. If your computer monitor has no HDMI input, then you'll need one of these adapters (This is around $7-10) :
    • HDMI to VGA Adapter - usually the VGA cable has blue connectors on the cable
    • HDMI to DVI Adapter - usually the DVI cable has white connectors on the cable

If you don't have a keyboard, mouse, or monitor around from an older computer, check yard sales or a thrift store like Goodwill.
