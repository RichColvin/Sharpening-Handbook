Calculators for the Sharpener
Web-based Calculators
  • Knife Sharpening Angle Calculator - CB created this tool which can be used to calculate the settings you will need to use with the knife jigs for a given angle on a knife blade.

  • Projection Calculator - use to calculate the projection needed on the knife jigs for a given angle on a knife blade. (If you care, the math behind this is documented in this link.)

  • Fly Cutter Projection Calculator - use to calculate the projection needed for a given nose angle on a fly cutter for the Rose Engine lathe. Use with the fly cutter adapter for the SVD-185/186 jig.
Computer-based &
Tablet-based Calculators
  • KnifeGrinders - Designed to work with the Front Vertical Base they sell. Unfortunately, with the far too early loss of Dr. Vadim Kraichuk, I do not think this is available any more.

  • TormekCalc2 - JVH (from the Tormek forum) created this tool which is an Excel spreadsheet that offers exact calculations and many advanced functions related to knife or tool grinding.

  • Polishing Angle Calc - This is a supplement to TormekCalc2 that was published by Gilles on the Tormek forum.
Other Web-Based Calculators
  • Turning Formulas Calculator - use to convert RPM to SFM, IPR, etc. (and vice versa)

  • Machinist's Calculators - use for the lathe, drill, and milling machine. I have found three variations, so you can use the one you prefer :
    • Little Machine Shop's Machinist's Calculator (I like working with these guys, so I generally use this one.)
    • I-logic Machinist's Calculator
    • Theoretical Machinist's Calculator
  • Calculator Edge - Calculators for about everything

  • Engineers' Edge has some useful tables and calculators. A few are listed below
    • Brinell and Rockwell hardness conversion.
    • Comparative Hardness Scales for Steel Table.
    • Mechanical Design, Manufacturing, and Engineering Calculators and Equations - a WHOLE LOT of web-based calculators.
