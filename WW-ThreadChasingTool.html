Thread Chasing Tool Set  

Side View, showing an exaggerated hollow grind

The cutters are sharpened to have a hollow grind on the top of the tool, behind the thread cutting area. This hollow grind is shown in an exaggerated picture below.

The tool is then honed by rubbing a 150+ grit diamond file against the top of the tool. The area to be honed is shown in the pictures as the area marked red.

Woodworkers are knotty.
