Edge Angle

McLeod tool

This tool is used in firefighting for cleaning up burnt areas. Parks Service use this tool for clearing up areas for hiking trails.

Guidelines shown below are for Included Angles (α).

A 6" smooth cut machinist’s hand file is useful to carry with you. Use it with a draw filing approach to resharpen the shovel as needed throughout the tool's usage for the day.

This could be up to 40°. Best to not sharpen the tines, though they should be straightened if bent.
Landscaping 30°

This could be up to 40°, but if a larger angle is needed, it is best to start at 35°. The landscaper may be hot and sweaty, but they are not wearing the clothing like a firefighter, so they will surely be expected to better care for the tool.

As above, it is best to not sharpen the tines, though they should be straightened if bent.
